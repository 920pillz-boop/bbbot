# 🤖 Agency Telegram Bot

Telegram-бот для модельного агентства с анкетированием, реферальной программой и встроенной админ-панелью.

## ⚙️ Установка

### 1. Требования
- Python 3.11+
- pip

### 2. Клонируйте / распакуйте проект
```bash
cd telegram-bot
```

### 3. Установите зависимости
```bash
pip install -r requirements.txt
```

### 4. Настройте конфигурацию
```bash
cp .env.example .env
```
Откройте `.env` и заполните:

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен бота от [@BotFather](https://t.me/BotFather) |
| `ADMIN_CHAT_ID` | Ваш Telegram ID (узнать у [@userinfobot](https://t.me/userinfobot)) |
| `CHANNEL_LINK` | Ссылка на ваш Telegram-канал |
| `WEBSITE_LINK` | Ссылка на сайт агентства |
| `DB_PATH` | Путь к файлу БД (по умолчанию `bot.db`) |

### 5. Запустите бота
```bash
python bot.py
```

## 📋 Функционал

### Для пользователей
| Кнопка | Действие |
|---|---|
| 📋 Профиль | Просмотр анкеты + редактирование каждого поля |
| 🤝 Рефералы | Реферальная ссылка, статистика, баланс |
| 📢 Наш канал | Ссылка на Telegram-канал |
| 🌐 Наш сайт | Ссылка на сайт |

### Анкета (9 полей)
1. Имя и фамилия
2. Рост (см)
3. Вес (кг)
4. Модель телефона
5. Соцсети
6. Место проживания
7. Личные ограничения
8. Желаемый уровень заработка
9. Опыт работы
10. Цели

### Реферальная программа
- Уникальная ссылка: `https://t.me/botname?start=ref_TELEGRAMID`
- **1-й уровень:** 5% от дохода приглашённой модели
- Начисление через `add_ref_bonus()` в `database.py`

### Статусы анкеты
| Статус | Описание |
|---|---|
| `new` | Новый пользователь |
| `filling` | Заполняет анкету |
| `reviewing` | Анкета на рассмотрении |
| `approved` | Одобрена |
| `rejected` | Отклонена |
| `active` | Активная модель |

## 🔧 Админ-панель

Доступна только пользователю с `ADMIN_CHAT_ID`:
- По кнопке **🔧 Админка** в главном меню
- По команде `/admin`

### Возможности
- 📊 **Статистика** — количество анкет по статусам
- 🆕 **Новые заявки** — список со статусом `reviewing`
- 📋 **Все модели** — полный список с пагинацией (по 5)
- ✅ **Одобренные** / 🟢 **Активные** — отдельные фильтры
- Просмотр полной анкеты любой модели
- Смена статуса: ✅ Одобрить / ❌ Отклонить / 🟢 Активировать
- Автоматическое уведомление модели при смене статуса

## 💰 Начисление бонусов

Когда модель получает доход — вызывайте функцию:

```python
from database import add_ref_bonus, update_user

# Начислить доход модели
await update_user(model_tg_id, income=new_total_income)

# Начислить реферальный бонус пригласившему
referrer_tg_id = user["ref_by"]  # из таблицы users
if referrer_tg_id:
    bonus = income_amount * 0.05  # 5%
    await add_ref_bonus(referrer_tg_id, model_tg_id, bonus)
```
